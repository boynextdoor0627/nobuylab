# Icon Usage Notes

`assets/icons/` 来自 `nobuylab_icon.zip`，当前已同步到 HTML 原型和微信小程序 MVP。

## 已接入位置

- HTML 底部导航：发现、冷静车、想买、记录、我的。
- HTML 搜索框、实验提醒、省钱广场头图。
- HTML 首页“今天先暂停哪一种冲动”选项。
- 小程序 MVP 今日暂停选项。
- 小程序 MVP 省钱广场动态封面和帖子详情封面。

## 当前映射

- `search.png`：发现、搜索。
- `wallet.png`：冷静车、旅行/提问类动态。
- `no-money.png`：想买暂停、种草灭火。
- `hourglass.png`：记录、实验提醒。
- `treasure-chest.png`：我的。
- `savings.png`：省钱广场、还好没买。
- `smartphone.png`：数码冲动。
- `wallet-passes-app.png`：穿搭冲动。
- `refund.png`：护肤/退一步思考。
- `bank.png`：家居/预算感。
- `money-bag.png`：其他冲动、本地动态默认图标。

## 后续建议

- 正式微信小程序 tabBar 图标建议单独导出 81x81 PNG，并控制体积。
- 带空格或括号的文件名暂时不在代码中引用，后续可统一改为无空格命名。
- Logo 与 App Icon 建议单独确认一张主视觉，不建议直接从功能 icon 中随意替换。
