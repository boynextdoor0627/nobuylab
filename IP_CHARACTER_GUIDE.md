# 不买研究所 IP 角色使用规范

## 核心识别

- 主体始终是绿渐变购物袋，保留白色提手、叉叉眼和弧形笑脸。
- 手脚使用黑色短肢体与厚底鞋，比例短而有力量感，保持可爱但不过度幼态。
- 角色不承担复杂说明，主要负责欢迎、暂停、完成和交流四种情绪反馈。

## 现有角色

- `mascot-wave.png`：挥手欢迎。用于启动页、新手引导和空状态。
- `mascot-pause.png`：伸手暂停。用于今日暂停、冷静箱和购买前提醒。
- `mascot-success.png`：跳跃庆祝。用于省下金额、连续记录和任务完成。
- `mascot-community.png`：托起三点气泡。用于广场、评论和分享场景。

## 后续延展

- 新表情优先改变姿势、手持物和眼睛状态，不改变购物袋轮廓与绿渐变。
- 单个页面最多出现一个大角色；列表和操作区只使用互动小图标，避免品牌形象挤占内容。
- 动画建议保持 0.6-1.2 秒：轻跳、挥手、眨眼或气泡弹出，不做持续晃动。
