# 不买研究所 NO BUY LAB

围绕“今天先不买一下”、冷静记录、消费冲动复盘与省钱社区的移动端 HTML 交互原型。

## V11.4 当前版本

- 默认启用极简日用模式，第一屏只保留“今天先不买什么”
- 第一屏改为选项优先；文字和金额都是可选项，用户不输入也能直接生成一条冷静记录
- 首页新增 7 日暂停点阵，强化轻量回访，而不是堆功能
- 默认底部导航收敛为：首页、想买、记录
- 商品库、省钱广场、挑战、档案和付费研究室退到完整模式
- 冷静记录保留轻量复盘，不再在默认成功页堆叠额外建议
- 保留完整模式，可一键查看省钱广场、每日清单、32 个品类和挑战功能
- 增加小程序迁移准备入口
- 增加商品图片本地 SVG 兜底，外链图片失败时仍可展示
- 导出数据版本号更新为 `V11.4`

## V11 品牌升级

## 产品规模

- 32 个消费品类
- 256 个原创虚构研究对象
- 中英文混合虚构品牌体系
- 每日清单、季节专题、动态推荐、连续挑战、每周上新
- 省钱广场 V1、模板投稿、收藏互动与优惠权益决策原型
- 冷静期复盘边界，区分真实复盘与原型演示复盘
- 隐私与本地数据说明弹窗
- 小程序迁移准备清单
- 本地商品图兜底
- 简洁模式 / 完整模式双模式体验

## 运行

直接打开 `index.html` 即可。上传 GitHub Pages 时请同时保留完整的 `assets` 文件夹。

## 说明

当前是纯前端原型，数据保存在浏览器本地；没有真实支付、交易、账号、公开发帖或真实优惠券接口。图片许可详见 `IMAGE_SOURCES.md`。

## 后续迁移建议

- 微信小程序阶段建议将当前单页原型拆为“发现、冷静车、实验、记录、我的、省钱广场、省钱研究室”等页面。
- 如果后续还要开发 App，可优先考虑 uni-app / Vue 3 路线，保留迁移到微信小程序、H5 和 App 的空间。
- 接入账号、云同步、AI 建议或付费功能前，应先补充隐私政策、用户授权、数据删除、内容审核和建议边界。
