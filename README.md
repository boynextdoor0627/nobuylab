# 不买研究所 · NoBuyLab

**不买研究所（BUMAI LAB）** 是一个面向年轻用户的“虚拟下单 + 冷静实验”交互原型。

用户可以把想买的东西放进“冷静车”，完成一次不产生真实交易的虚拟下单，在冷静期结束后确认自己是否真的购买，并记录“确认守住”的金额。

> 不是不让你买，是帮你先想一想。

## 当前版本

当前仓库为 **V3 纯前端 HTML 交互原型**，主要用于产品验证、UI/UX 测试和开发沟通。

已实现：

- 研究大厅与商品浏览
- 冷静车与虚拟结算
- 冷静实验与虚拟配送
- 冷静记录与结果确认
- 用户自定义消费冲动
- 消费习惯洞察与实验报告
- “省钱研究室”未来付费功能原型
- 浏览器本地数据保存

尚未接入：

- 用户账号与云端同步
- 服务器、数据库与管理后台
- 真实 AI 模型
- 真实支付
- 微信小程序或原生 App

## 在线体验

部署 GitHub Pages 后，可通过仓库的 Pages 地址在线访问。

## 本地运行

这是一个单文件静态项目，不需要安装依赖。

1. 下载或克隆仓库。
2. 双击 `index.html`。
3. 使用 Chrome、Edge 或 Safari 打开。

也可以在项目目录启动一个本地静态服务器：

```bash
python3 -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

## GitHub Pages 部署

1. 打开仓库 `Settings`。
2. 进入 `Pages`。
3. 在 `Build and deployment` 中选择 `Deploy from a branch`。
4. 选择 `main` 分支和 `/(root)` 目录。
5. 保存并等待部署完成。

## 产品原则

- 所有订单均为虚拟体验，不产生真实交易。
- 不使用真实电商平台或品牌 Logo 进行混淆性模仿。
- 不鼓励攀比式虚拟消费。
- “模拟消费金额”与“确认守住金额”分开统计。
- 用户财务与消费数据不用于诱导消费。
