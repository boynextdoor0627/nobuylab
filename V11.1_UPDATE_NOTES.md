# 不买研究所 V11.1 更新说明

## 小程序迁移准备

- 在“我的研究档案”增加“小程序迁移准备”入口。
- 明确后续可拆分页面：发现、冷静车、实验、记录、我的、省钱广场、省钱研究室。
- 明确后续可拆分数据模块：商品库、广场内容、冷静订单、每日清单、挑战、省钱报告。
- 补充上线前需要处理的隐私、授权、内容审核、付费功能和 AI 建议边界。

## 原型稳定性

- 页面标题、描述、品牌标签与导出数据版本号更新为 V11.1。
- 商品图片优先使用外链场景图；外链失败时自动回退到本地 `assets/products` SVG 素材。
- 订单列表小图同样增加本地兜底，减少 GitHub Pages 或小程序预览时的空图情况。

## 版本定位

V11.1 仍然是纯前端 HTML 原型，不接入账号、云同步、真实支付、公开发帖或真实优惠券接口。
