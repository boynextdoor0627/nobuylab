# 不买研究所 V11 更新说明

## 全新绿色品牌体系

- 主色由赤陶橙调整为鲜绿色、荧光青柠与深森林绿。
- App 背景改为米白与浅薄荷绿，减少传统电商平台的视觉联想。
- 按钮、标签、导航、分享报告、省钱广场与优惠模块统一使用绿色品牌语言。

## 叉叉眼笑脸 IP 正式进入产品

- 重绘可缩放 Logo：绿色购物袋、叉叉眼、笑脸与荧光星标。
- 首页数据英雄卡加入 3D IP 形象。
- 首次启动引导加入 3D 吉祥物。
- 省钱广场、个人中心、App 图标和浏览器 favicon 使用统一 IP 标识。
- 品牌英文名统一为 `NO BUY LAB`。

## 交互与兼容

- 保留 V10 全部功能、256 个研究对象、省钱广场和本地数据结构。
- 不改变原有主要操作流程，旧版本本地数据仍可继续使用。
- 手机端保持单页 HTML 直接运行，GitHub Pages 可直接部署。
- 增加冷静期复盘边界：未到期时先提示等待，原型演示复盘会明确标注。
- 增加隐私与数据弹窗，说明冷静记录、私人商品、省钱研究室资料和广场草稿仅保存在本机浏览器。
